# 🐕 nextjs-tailwind-starter

A starter for Next.js 13.4+ with preconfigured Tailwind CSS, ESLint, VSCode Settings and Prettier.

```bash
# install
yarn

# run dev server
yarn dev
yarn dev --turbo

# build
yarn build
yarn buuld --turbo

# run production server
yarn start

# lint
yarn lint

# prettier
yarn prettier

# clean
yarn clean

# reinstall
yarn reinstall
```
